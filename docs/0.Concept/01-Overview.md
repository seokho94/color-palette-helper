# 색상 추천 서비스 개요

## 프로젝트 목적

웹 서비스 개발 시 UI 디자인을 위한 색상 조합을 자동으로 추천하는 서비스

## 주요 목표

1. **기존 서비스 분석**: 웹사이트의 컨셉과 색상 팔레트 자동 분석
2. **지능형 추천**: 색상 이론 및 트렌드 기반 조화로운 색상 조합 제안
3. **접근성 보장**: WCAG 기준 준수 여부 자동 검증
4. **실용성**: 개발자/디자이너가 즉시 사용 가능한 색상 코드 제공

## 타겟 사용자

- UI/UX 디자이너
- 프론트엔드 개발자
- 제품 기획자
- 스타트업 창업자 (디자인 리소스 부족)

## 핵심 차별점

- 단순 팔레트 생성이 아닌 **컨셉 기반 추천**
- **실시간 UI 컴포넌트 미리보기**
- **접근성 자동 검증**
- **산업별 맞춤 추천**
