# 핵심 기능 요구사항

## 1. 기존 서비스 분석

### 1.1 웹사이트 분석
- **입력**: 웹사이트 URL
- **처리**:
  - 스크린샷 자동 캡처
  - 주요 색상 자동 추출
  - 색상 사용 비율 분석
- **출력**:
  - 주요 색상 팔레트 (Primary, Secondary, Accent)
  - 색상 분포 차트
  - 컬러 히스토그램

### 1.2 이미지 분석
- **입력**: 이미지 파일 업로드 (PNG, JPG, SVG)
- **처리**:
  - 지배적인 색상 추출 (Dominant Colors)
  - 브랜드 로고에서 주요 색상 식별
  - 색상 클러스터링
- **출력**:
  - 추출된 색상 팔레트
  - 각 색상의 사용 비중

### 1.3 수동 입력
- Color Picker를 통한 직접 색상 선택
- HEX, RGB, HSL 코드 직접 입력

## 2. 색상 조합 추천

### 2.1 색상 조화 규칙 기반
- **보색 (Complementary)**: 색상환에서 180° 반대편
- **유사색 (Analogous)**: ±30° 인접 색상
- **삼색 조화 (Triadic)**: 120° 간격 3색
- **사색 조화 (Tetradic)**: 90° 간격 4색
- **분할 보색 (Split-Complementary)**: 보색의 양옆 색상

### 2.2 무드/감성 기반 추천
| 무드 | 특징 | 색상 예시 |
|------|------|-----------|
| Modern | 세련됨, 미래지향적 | 블루, 그레이, 화이트 |
| Classic | 전통적, 우아함 | 네이비, 골드, 크림 |
| Minimal | 단순함, 깔끔함 | 블랙, 화이트, 베이지 |
| Vivid | 활기찬, 에너제틱 | 레드, 옐로우, 오렌지 |
| Natural | 자연스러움, 편안함 | 그린, 브라운, 스카이블루 |
| Luxury | 고급스러움 | 블랙, 골드, 화이트 |

### 2.3 산업별 트렌드
- **핀테크**: 신뢰감 (블루 계열)
- **헬스케어**: 안정감 (그린, 화이트)
- **식음료**: 식욕 유발 (레드, 오렌지, 옐로우)
- **교육**: 집중력 (블루, 그린)
- **패션**: 트렌디함 (시즌별 변동)

### 2.4 접근성 검증
- **WCAG 2.1 대비 비율 자동 계산**
  - AA 기준: 본문 4.5:1, 큰 글자 3:1
  - AAA 기준: 본문 7:1, 큰 글자 4.5:1
- **색맹 시뮬레이션** (Protanopia, Deuteranopia, Tritanopia)
- **자동 대안 색상 제안** (기준 미달 시)

## 3. 인터랙티브 도구

### 3.1 색상 팔레트 생성기
- 베이스 컬러 선택 → 자동 조화 색상 생성
- 색상 잠금(Lock) 기능: 특정 색상 고정 후 나머지만 재생성
- 명도/채도 조절 슬라이더
- 랜덤 팔레트 생성

### 3.2 실시간 UI 미리보기
- **컴포넌트별 적용**:
  - 버튼 (Primary, Secondary, Disabled)
  - 카드 (Background, Border, Shadow)
  - 폼 (Input, Focus, Error, Success)
  - 네비게이션 (Header, Footer, Menu)
  - 타이포그래피 (Heading, Body, Link)
- **다크모드 자동 생성**
- **모바일/데스크톱 미리보기**

### 3.3 색상 코드 변환 & 내보내기
- **지원 포맷**:
  - HEX (#FF5733)
  - RGB (rgb(255, 87, 51))
  - HSL (hsl(9, 100%, 60%))
  - RGBA / HSLA (투명도 포함)
  - CMYK (인쇄용)
- **내보내기 포맷**:
  - CSS Variables
  - SCSS Variables
  - Tailwind Config
  - JSON
  - Figma Plugin 연동

### 3.4 히스토리 및 즐겨찾기
- 생성한 팔레트 자동 저장
- 즐겨찾기 기능
- 팔레트 공유 (URL 생성)
- 프로젝트별 분류

## 4. 부가 기능

### 4.1 그라디언트 생성기
- 2색 이상 그라디언트 자동 생성
- 방향 조절 (Linear, Radial)
- CSS 코드 자동 생성

### 4.2 색상 트렌드 분석
- 최근 인기 색상 조합
- 연도별 트렌드 차트
- 산업별 인기 색상

### 4.3 AI 기반 추천 (선택 사항)
- 브랜드 키워드 입력 → AI가 색상 조합 제안
- 경쟁사 분석 → 차별화된 색상 추천
- 사용자 피드백 학습 → 개인화 추천
