# 참고 자료

## 1. 기존 색상 도구 분석

### 1.1 주요 경쟁 서비스

#### [Coolors.co](https://coolors.co)
- **강점**:
  - 스페이스바로 즉시 재생성 (빠른 UX)
  - 색상 잠금(Lock) 기능
  - iOS/Android 앱 제공
  - 트렌드 팔레트 갤러리
- **약점**:
  - 산업별 추천 부재
  - 접근성 검증 기능 약함
- **배울 점**: 직관적인 키보드 단축키

#### [Adobe Color](https://color.adobe.com)
- **강점**:
  - 색상환 기반 시각적 조작
  - Adobe Creative Cloud 연동
  - 트렌드 및 라이브러리 기능
- **약점**:
  - UI가 복잡함 (초보자 진입장벽)
  - 웹사이트 분석 기능 없음
- **배울 점**: 색상 조화 시각화

#### [Paletton](https://paletton.com)
- **강점**:
  - 고급 색상 이론 적용
  - 색맹 시뮬레이션
  - UI 예제 미리보기
- **약점**:
  - 오래된 디자인
  - 모바일 최적화 부족
- **배울 점**: 색맹 시뮬레이션 정확도

#### [Khroma](http://khroma.co)
- **강점**:
  - AI 기반 개인화 추천
  - 사용자 선호도 학습
  - 무한 스크롤 팔레트
- **약점**:
  - 초기 학습 필요 (50개 선택)
  - 무료 기능 제한적
- **배울 점**: AI 학습 방식

#### [ColorSpace](https://mycolor.space)
- **강점**:
  - 그라디언트 자동 생성
  - 3색 팔레트 추천
  - CSS 코드 즉시 복사
- **약점**:
  - 옵션이 제한적
  - 고급 기능 부족
- **배울 점**: 그라디언트 생성 알고리즘
