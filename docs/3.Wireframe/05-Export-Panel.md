## 5. Copy 버튼 상태

```

│      [Copy Code]             │

```

### 5.2 복사 성공

```
┌──────────────────────────────┐
│      [✓ Copied!]             │  ← 2초간 표시
└──────────────────────────────┘
```

```
┌──────────────────────────────┐
│      [✗ Failed]              │

+ Toast 알림: "Failed to copy. Try again."
```
