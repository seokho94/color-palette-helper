## 🎨 디자인 원칙

### 1. 미니멀리즘
- 불필요한 장식 제거
- 사용자 생성 색상이 주인공
- 모노톤 베이스

### 2. 명확성
- 3번의 클릭 내 목표 달성
- 즉각적인 피드백
- 명확한 라벨

### 3. 접근성
- WCAG AA 준수 (UI 자체도)
- 키보드 네비게이션 지원
- Focus indicator 명확
