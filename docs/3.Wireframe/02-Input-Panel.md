## 7. 인터랙션 상세

### 7.1 Accordion 토글

```javascript
// 클릭 시
onClick={() => {
  setExpanded(!expanded)
}}

// 애니메이션
transition: height 0.3s ease-out
```

### 7.2 Color Picker 드래그

```javascript
// Hue wheel 클릭/드래그
onMouseDown / onMouseMove
→ Hue 업데이트
→ Debounce 100ms
→ 팔레트 재생성
```

### 7.3 이미지 업로드

```javascript
// Drag & Drop
onDrop={(e) => {
  const file = e.dataTransfer.files[0]
  uploadImage(file)
}}

// File Input
<input type="file" accept="image/*" />
```
