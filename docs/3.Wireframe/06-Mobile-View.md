## 8. Touch Gestures

### 8.1 Swipe (좌우)
```
Preview 탭 전환
Palette 색상 스크롤
```

### 8.2 Long Press
```
색상 카드 → 옵션 메뉴
(Edit, Lock, Copy, Delete)
```

### 8.3 Pull to Refresh
```
메인 화면 → 랜덤 팔레트 생성
```

## 10. 모바일 최적화

### 10.1 Touch Target

```
최소 크기: 44px × 44px

[Button] → 48px height
[Input]  → 48px height
[Card]   → 60px min height
```

### 10.2 스크롤 영역

```
각 섹션 독립 스크롤
Bounce 효과 (iOS)
Scroll Snap (색상 카드)
```

### 10.3 성능

```
이미지 최적화: WebP
Lazy Loading: Preview
Virtual Scroll: History (많을 경우)
```
